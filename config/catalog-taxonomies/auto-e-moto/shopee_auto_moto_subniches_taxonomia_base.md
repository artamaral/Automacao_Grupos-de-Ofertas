# Taxonomia base — Shopee Auto e Moto
Arquivo oficial recomendado para execução automática: `shopee_auto_moto_subniches_taxonomia_base.json`.
## Regras fechadas
- Não criar novos subnichos durante a limpeza.
- Novas palavras-chave podem ser adicionadas somente se apontarem para um subnicho já existente em `allowed_subniches`.
- Termos sem mapeamento devem ir para `unmapped_source_keywords` e receber `acessorios-geral` como fallback.
