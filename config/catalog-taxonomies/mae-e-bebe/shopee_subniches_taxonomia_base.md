# Taxonomia base — Subnichos Shopee Mãe e Bebê

Este arquivo é a lista fechada de `subniches` permitidos e suas respectivas palavras-chave. Novos `subniches` não devem ser criados. Novas palavras-chave podem ser adicionadas somente se forem associadas a um `subniche` já existente nesta lista.

## Regras de uso

1. Usar `source_hits_keywords` por correspondência exata após normalização.
2. Usar `productName_fallback_keywords` somente quando `source_hits` for genérico ou vazio.
3. Quando nenhuma regra casar, usar `bebe-geral` e registrar o termo não mapeado no resumo.
4. Não criar novo `subniche` durante a limpeza.
